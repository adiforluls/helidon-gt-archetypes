# Helidon Archetypes for static content support

```bash
Clone the github repository, then shift to root directory and run the following command:
mvn install

Now shift to a new directory, and run the following command:
mvn archetype:generate -DinteractiveMode=false -DarchetypeGroupId=io.helidon.archetypes -DarchetypeArtifactId=helidon-gt-se -DarchetypeVersion=2.0.1 -DgroupId=io.helidon.examples -DartifactId=justsee -Dpackage=mypackage

Note: the last 3 parameters are variables, you can change them

Your project has been generated!
```
